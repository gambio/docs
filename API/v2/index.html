    Gambio GX3 API 3.9.0¶

    This is the Gambio GX Restful API documentation for the GX shop software. You can find out more about Gambio at http://www.gambio.de.

    address_formats¶

    GET /address_formats¶

    Get Address Formats

    Description

    Get all address formats or a single address format.

    Input parameters

    Parameter In Type Default Nullable Description
    basicAuth header string N/A No
    fields query None No *Optional parameter*. If provided limits the properties to be included in the response.
    page query None No *Optional parameter*. If provided, displays the specified page. Defaults to 0.
    per_page query None No *Optional parameter*. If provided, limits the amount of records in the response. Defaults to 50.
    search query None No *Optional parameter*. Search for a url-encoded keyword.
    sort query None No *Optional parameter*. Sorts by the desired properties. Maximum sorting fields are 5. (prefix property with + for ascending, - for descending)

    Response 200 OK

    Response 400 Bad Request

    Response 404 Not Found

    attachments¶

    POST /attachments¶

    Upload Attachment

    Description

    If an email contains an attachment this must be uploaded before the email is sent. This method accepts the upload of one file at a time. It will return its temporary path which can be used as the attachment path in the email JSON data. The name of the file form field is not taken into concern (can be whatever). The important rule is that only one file will be uploaded at a time and that you are sending the Content-Type: multipart/form-data header instead of application/json.

    Input parameters

    Parameter In Type Default Nullable Description
    basicAuth header string N/A No
    filename formData None No The final filename
    upload_attachment formData None No The attachment to upload

    Response 201 Created

    Response 400 Bad Request
